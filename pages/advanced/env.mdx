# .ENV explanation

## PORT

Port on which lcs-cloud-storage server API will run

```env
PORT=9110
```

## HOST

Host on which lcs-cloud-storage server API will run

```env
HOST="localhost"
```

## SECRET_KEY

Secret key for jwt to sign authentication tokens

```env
SECRET_KEY="supersecret"
```

## DB_NAME

Name for database where will be stored signed up users

```env
DB_NAME="lcs_user_repository"
```
