# Server configuration

lcs cloud storage provides a cli for easy server configuration.

## Config command

```shell
config|cfg [options]    define a default config for the server
Options:
--dhost <dhost>    define default server host
--dport <dport>    define default server port
--jwtkey <jwtkey>  define jwt key for signing login tokens
--authexp <time>   define a life time for login sessions. Format: "2 days", "10h", "7d"
-h, --help         display help for command
```

## About

### dhost

Default host for the server, which it will use.

**Usage examples**

```shell
$ lcs config --dhost=localhost
```

```shell
$ lcs config --dhost localhost
```

### dport

Default port for the server, on which it will work.

**Usage examples**

```shell
$ lcs config --dport=3000
```

```shell
$ lcs config --dport 3000
```

### jwtkey

‼️_**Required for lcs server. If it's not provided it will throw error on login or trying to reach protected route!**_

Defines a jsonwebtoken secret key to sign authentication tokens.

**Usage examples**

```shell
$ lcs config --jwtkey=superkey
```

```shell
$ lcs config --jwtkey superkey
```

### authexp

Defines a lifetime of authentication session for users

**Usage examples**

```shell
$ lcs config --authexp "2 days"
```

```shell
$ lcs config --authexp 7h
```

```shell
lcs config --authexp 10d
```
