# Command Line Interface (CLI)

Usage: `lcs [options] [command]`

#### Options:

```shell
-V, --version    output version number
-l, --log        enable logging
-h, --help       display help for command
```

#### Commands:

```shell
config|cfg [options]    define a default config for the server
Options:
--dhost <dhost>    define default server host
--dport <dport>    define default server port
--jwtkey <jwtkey>  define jwt key for signing login tokens
--authexp <time>   define a life time for login sessions. Format: "2 days", "10h", "7d"
-h, --help         display help for command
```

```shell
server|run [options]    start the local cloud storage server
Options:
-p, --port <port>       tell server which port to use
-h, --host <host>       tell server which host to use
-https, --secure [days] tell server to use https protocol
--help                  display help for command
```

```shell
prune [options]    clear server data and delete it
Options:
-c, --config    clear server config
-s, --storage   clear server storage
-db, --database drop user database
-l, --logs      delete server logs
-a, --all       clear all server data
-h, --help      display help for command
```
