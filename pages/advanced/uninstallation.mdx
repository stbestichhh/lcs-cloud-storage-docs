# Uninstallation

## Clear server data

* To clear server config

  ```shell
  $ lcs prune --config
  ```

* To clear server storage

  ```shell
  $ lcs prune --storage
  ```

* Delete all server data

  ```shell
  $ lcs prune --all
  ```

## Delete server package

```shell
$ npm uninstall -g lcs-cloud-storage
```
