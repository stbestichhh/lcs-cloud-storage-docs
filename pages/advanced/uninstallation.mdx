# Uninstallation

## Clear server data

* To clear server config

  ```shell
  $ lcs prune --config
  ```

* To clear server storage

  ```shell
  $ lcs prune --storage
  ```

* To clear users database

  ```shell
  $ lcs prune --database
  ```

* To delete server logs
  ```shell
  $ lcs prune --logs
  ```

* Delete all server data

  ```shell
  $ lcs prune --all
  ```

## Delete server package

```shell
$ npm uninstall -g @lcs-cloud-storage/server
```
