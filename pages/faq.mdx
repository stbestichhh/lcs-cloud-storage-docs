# FAQ

## Where is the config file?

The roor directory located in home directory of your os.

| **Platform** | **Path** |
| -------- | ---- |
| **Linux/MacOS**| `~/.config/lcs_cloud_storage/lcs.config` |
| **Windows** | `C:\Users\username\.config\lcs_cloud_storage\lcs.config` |

## Where is server root directory?

The roor directory located in home directory of your os. There will be users database, logfile and 'storage' directory.

| **Platform** | **Path** |
| -------- | ---- |
| **Linux/MacOS**| `~/.lcs_cloud_storage` |
| **Windows** | `C:\Users\username\.lcs_cloud_storage` |
